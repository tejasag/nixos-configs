# `hydrogen`

```
H (or Hydrogen) is a CLI with a bunch of utilities.

Usage:
  h [command]
  h [command] --help | -h
  h --version | -v

Commands:
  rebuild   Rebuild NixOS or home-manager
  update    Update flake inputs
  test      Rebuild and test the system
  search    Search for a package in nixpkgs
```
